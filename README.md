# Quiz

A quiz template based on **React** + **Vite** + **Socket.io** 

### Requirements

- [Node.js](https://nodejs.org/)

### Setup

`git clone https://github.com/vanorium/quiz.git`

`cd quiz`

`npm install`

### Preview

1. Firstly, enabling the server (initially it will be started localy):

- `cd server`

- `npm run start`
